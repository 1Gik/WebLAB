# Лабораторні роботи з веброзробки

Цей репозиторій містить виконання лабораторних робіт 1–4 зі створення клієнт-серверного вебзастосунку.

Проєкт являє собою лендінг майстерні з відновлення старої електроніки з формою зворотного зв’язку, яка працює через REST API.

---

## Технології

### Frontend
- HTML5
- CSS3 (з підтримкою темної теми)
- Vanilla JavaScript

### Backend
- Python 3
- Flask (REST API)

### Інфраструктура
- Docker
- Docker Compose

---

## Як запустити проєкт (Лабораторна 4)

Для запуску проєкту на комп’ютері мають бути встановлені Docker і Docker Desktop.

### 1. Склонуйте репозиторій

```bash
git clone <repo_url>
cd <project_folder>
```

### 2. Запустіть контейнер

```bash
docker compose up --build
```

### 3. Дочекайтеся запуску сервера

У терміналі має з’явитися повідомлення:

```text
Running on http://127.0.0.1:5000
```

---

## Як протестувати роботу

1. Відкрийте браузер та перейдіть за адресою:

```text
http://localhost:5000
```

2. Перевірте:
- анімації появи тексту;
- перемикач темної та світлої теми;
- форму зворотного зв’язку.

3. У секції **«Заявка на відновлення»**:
- заповніть форму;
- оберіть пункт **«Інше»** у списку — з’явиться додаткове поле;
- натисніть **«Відправити заявку»**.

Після цього дані будуть передані на Python-сервер без перезавантаження сторінки через `fetch`, а на екрані з’явиться повідомлення про успішну відповідь сервера.

---

## Як зупинити сервер

Для зупинки контейнера натисніть:

```bash
Ctrl + C
```

або виконайте команду:

```bash
docker compose down
```
